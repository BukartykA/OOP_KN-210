Завантаження конфігурації з файлу оточення за допомогою load_dotenv().

Автентифікація запиту до серверів Google.

Формування JSON-пакета з інструкціями для ШІ.

3. Технічні виклики та їх вирішення (Debugging) 🔍
Під час виконання роботи було виявлено та усунуто низку проблем:

Encoding Error: Виправлено UnicodeDecodeError при зчитуванні .env шляхом перекодування файлу в UTF-8.

Rate Limiting: Оброблено помилку 429 RESOURCE_EXHAUSTED (ліміти безкоштовного тарифу).

API Routing: Виправлено помилку 404 NOT_FOUND через зміну версій API (v1 / v1beta) та прямі HTTP-запити через requests.

💻 Програмний код (Стабільна версія)
Python
import os
import requests
from dotenv import load_dotenv

# Ініціалізація оточення 🔐
load_dotenv()
api_key = os.getenv("GOOGLE_API_KEY")

# Налаштування параметрів запиту 🌐
url = f"https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash:generateContent?key={api_key}"
headers = {'Content-Type': 'application/json'}
data = {
    "contents": [{"parts": [{"text": "Привіт! Напиши 'Агент KN-210 готовий'."}]}]
}

def run_agent():
    try:
        response = requests.post(url, headers=headers, json=data)
        if response.status_code == 200:
            print("✅ ВІДПОВІДЬ АГЕНТА:", response.json()['candidates'][0]['content']['parts'][0]['text'])
        else:
            print(f"⚠️ Помилка API: {response.status_code}")
    except Exception as e:
        print(f"❌ Помилка: {e}")

if __name__ == "__main__":
    run_agent()
